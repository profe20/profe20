Main File Structure

All the file are well organized, its so easy to work with the template.
1. Unzip the files.
2. Open "peulis" folder.
You will find the 404.html, about.html, blog.html, cart.html, checkout.html, contact.html, destination-four.html, destination-single.html, destination-three.html, faq.html, gallery.html, guides.html, index.html, login.html, product.html, register.html, single-blog.html, single-product.html, tour-details.html, tour-sidebar.html file on the root that will be edited.
There are 1 directory called "assets".
Also you will see 5 directories under assets directory "flaticon, css, js, fonts, img".
In the css folder Content you will find style.css and other styling files the all the other files expect the htmls.
Js folder contain only the jquery library. Fonts folder contain essetial font files for fonts.

Installing Template (FTP Install)

  • Unzip the archive downloaded from your Envato account
  • Open your FTP manager and login to your hosting.
  • Browse the required directory (Normally public_html)
  • Upload the files inside peulis folder.
